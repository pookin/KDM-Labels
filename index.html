Help

Search: Type in the search box to find KDM items like Fighting Arts, Philosophies, Disorders, etc.

Note: For Knowlages a checkmark will apear next to the last level you printed. When priting Philosophies it will use the last printed level for the tenent knowlage.

Menu (☰):

  • Label Settings: Adjust label dimensions and margins.
  • Content Filters: Select which expansions or philosophies to include/exclude from your search results.
  • Survivor Sheet: Opens a printable PDF of the KDM: Survivor Record Sheet. Make sure to print at actual size, you can print it double sided make sure to set it to flip on the short side.

Note: Ensure your browser's pop-up blocker is not blocking the print window or the Survivor Sheet PDF.

Tip: When printing, ensure your printer settings match the label size defined in "Label Settings" and that any "Scale to Fit" options are turned OFF for accurate sizing.

Printer and Label Suggestions

The survivor sheet was designed around 54mmx25mm or 2"x1" Labels.

  • Dymo 550 30336 Labels
    • 550 Printers need labels with RFID chips.
  • Dymo 450 30336 Labels
  • Rollo, Zebra, Munbyn, Nelko Labels

For printers you want a thermal printer that doesn't use an app to print. Typically looking for shipping label printers. I use a DYMO 4XL printer but these others should also work.

  • Dymo LabelWriter 550 Turbo
    • If you can get a 450 or 4XL they arent lock down on which labels they can use.
  • MUNBYN Label Printer
  • Nelko Label Printer
